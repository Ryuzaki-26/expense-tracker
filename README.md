# Expense Tracker Application

## Overview

Expense Tracker is a web application developed using HTML, CSS, and JavaScript that helps users manage daily expenses.

## Features

- Add Expense
- Edit Expense
- Delete Expense
- Search Expense
- Category Filter
- Budget Tracking
- Remaining Budget Calculation
- Local Storage Persistence

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage
